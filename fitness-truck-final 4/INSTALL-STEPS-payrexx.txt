1. In Supabase SQL Editor, run: SUPABASE-payrexx-payment-intents.sql
2. Replace these files in fitness-truck-final 4:
   - index.html
   - account.html
   - app.js
   - netlify/functions/_payment-helpers.js
   - netlify/functions/create-payrexx-payment.js
   - netlify/functions/payrexx-webhook.js
3. Push to GitHub
4. Let Netlify deploy
5. In Payrexx, create a webhook:
   - URL: https://fitnesstruck.ch/.netlify/functions/payrexx-webhook
   - Type: JSON
   - Retry on failure: ON
6. Test with a paid session

Notes:
- 0 CHF sessions still register directly without Payrexx.
- Paid sessions redirect to Payrexx checkout.
- Booking is only finalized when the webhook confirms a successful payment.
